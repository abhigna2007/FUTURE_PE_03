# FUTURE_PE_03 – SEO Content Cluster Generator

## 📌 Business Chosen
Glamour Studio – Premium salon located in Banjara Hills, Hyderabad.

## 🎯 Main Keyword
Best Salon in Banjara Hills

## ❗ Problem
Many business websites fail to rank on Google because:
- Blogs are unstructured
- Content does not match search intent
- No proper SEO strategy is followed

## 💡 Solution
This project uses AI-powered prompt engineering to generate a complete SEO content system, including:
- 1 Pillar Blog (main article)
- 4 Supporting Blogs (content cluster)
- Keyword targeting and search intent
- Internal linking strategy

## 🧠 SEO Strategy Used
- Focused on a primary keyword: *Best Salon in Banjara Hills*
- Created supporting blogs around related topics
- Applied local SEO (Hyderabad, Banjara Hills)
- Used structured headings (H1, H2, H3)
- Added internal linking for better ranking

## 🛠 Tools Used
- ChatGPT (for content generation)
- GitHub (for documentation and submission)

## 📂 Project Structure
- prompts/ → Contains SEO prompt used
- outputs/ → Contains generated blogs
- assets/ → Contains preview image

## 📊 Output Includes
- 1 detailed pillar blog (800–1000 words)
- 4 supporting blogs (300–500 words each)
- Keyword list
- Search intent explanation
- Internal linking suggestions
